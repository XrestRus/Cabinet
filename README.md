# Работа написаная на Laravel и Vue

## Использовалось:
- Laravel
- Vue
- Inertia.Js
- MySql

## Установка:
- Создать БД (laravel-server)
- Запустить в консоле команды:
    - composer install
    - php artisan migrate:flesh --seed
    - npm run hot
